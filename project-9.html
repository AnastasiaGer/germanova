Big trip

Description : «Big Trip» is a modern service for real travelers. The service helps to plan the trip route in detail, calculate the cost of the trip and get information about the sights. The minimalistic interface will not give a reason for distraction and will focus on travel planning. The application consists of two screens: «Trip itinerary» and «Statistics». A new waypoint is created by clicking on the «New Event» button. The application supports offline mode.
Tools : HTML5 / CSS3 / JavaScript / Webpack
Completion : June 2020
