What to watch

Description : "What to watch" is a new generation online cinema. Watch new items absolutely free and in the best quality. Leave reviews, rate and choose only the best from the world of big cinema. The application consists of several pages: Main (/), Sign In (/ login), MyList (/ mylist), Film (/ films /: id), Add review (/ films /: id / review), Player (/ player / : id). Pages «MyList», «Add review» are available only to authorized users. If the user is not logged in, then when going to these pages, a redirect to the «Sign In» page is performed. If the user is not logged in, then when trying to go to the private page, a redirect to the «Sign In» page (/ login) is performed. In the right corner of the header, the user's avatar (if the user is authorized) or the «Sign In» link (if the user is not authorized) is displayed. Clicking on the user's avatar takes you to the «MyList» page (/ mylist).
Tools : React / Redux / Typescript / Jest / Enzyme / Webpack
